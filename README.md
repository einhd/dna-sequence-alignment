# dna-sequence-alignment
 
1-) Install Phyton3
	
2-) Install the latest version of Flask via Console
	$ pip install Flask

3-) Install Numpy version 1.19.3 (Might not work with later versions, There is a bug in Windows runtime preventing the latest numpy version to work properly.)

4-) After all of these steps, Open the flaskApp folder as project.

5-) And Run the app.py program

6-) Once the compiling is done, you can see the program is running on localhost in the console.

7-) Open a browser tab and copy the localhost adress shown in terminal

-----------------------------------------------------------------------------------------------------------------------------------------

8-) The file containing the sequences you will upload should be in .txt format and not exceeding 5 lines.

9-) Everything written on the output screen is kept in the text file, and the text files are deleted and overwritten when the program is closed and restarted.
